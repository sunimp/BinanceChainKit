# BinanceChainKit.Swift

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/BinanceChainKit.Swift.git", .upToNextMajor(from: "2.1.0"))
]
```
